# 神烬使徒 / Apostles of God-Ash

《神烬使徒 / Apostles of God-Ash》是一款叙事向角色扮演游戏原型。当前课堂展示版聚焦第一章「低语田野」：玩家扮演「无音回响」，进入被旧神胃囊污染的田野，用骰子判定完成遭遇、对话、战斗与归档。

PV 与实机演示视频：

https://t.bilibili.com/1220737294506393657?share_source=pc_native

## 当前版本

当前同步版本是 `feature/card-duel-mvp` 上的课堂展示版，核心目标不是完整商业版，而是把世界观、视觉气质、骰子战斗和概念片式演出串成一条可展示流程。

已接入内容：

- 开场收藏家对话与演出。
- 低语田野入口演出。
- 空腹者、农夫、谷仓王遭遇前演出。
- 空腹者战后演出。
- 谷仓王场景入场、遭遇、失败与胜利结算演出。
- 玩家行动轮：攻击、重击、蓄防、大招。
- 敌方攻击轮：敌人攻击，玩家自动投防御骰。
- 重击判定：命中高出敌方防御 5 点后触发双倍伤害。
- 蓄防判定：下一次防御 D20 投两次取最高，并可反弹。
- 大招判定：攻击累计后可用，投 D6 无视防御造成直伤。
- 圣匣日志、掉落奖励、角色立绘、骰子 UI 和技能图标的展示版接入。

## 世界观

旧神已死，世界只剩回响。

宇宙仿佛某种巨大的乐器，不断释放一种频率。旧神曾经用自己的身体过滤这种频率，把它转化成神力，再把神力转化成统治秩序。

后来宇宙产生了自我意识。理性出现的第一刻并没有带来稳定，反而让原本可被过滤的频率变得崩乱。旧神承载不住这种变化，身体爆裂，只剩器官、残骸和执念散落人间。

低语田野是被旧神胃囊影响的地方。胃囊象征贪婪、食欲和吞噬，所以这片田地不再生产粮食，而开始吞噬人。谷仓王曾经是试图打开粮仓拯救百姓的英雄，但他献祭自己后，反而成了田野继续收割众人的代表。

玩家要终止田野的吞噬，也要承受神的胃囊。问题随之出现：当一个人把旧神残骸吸收到身体里，他还是原来的自己吗？

## 游戏类型

当前定位更接近叙事角色扮演游戏，而不是传统卡牌游戏。它的战斗机制借鉴桌面 RPG 的骰子判定：玩家不是简单点按钮打怪，而是在每次行动中提交一次「证词」，由骰子决定这份证词是否能穿过对方的抵抗。

可以理解为：

- 叙事 RPG 的角色、世界观和选择节奏。
- 骰子驱动的命中、防御、重击和大招判定。
- 关卡推进采用固定章节顺序，当前不是随机地图。
- 牌组构筑、局外成长和更复杂的 BD 系统仍是后续方向。

## 当前边界

这是课堂展示与垂直切片版本，不是完整游戏。

暂时不主攻：

- 完整十城邦内容。
- 大型随机地图。
- 商店和长期局外成长。
- 多结局与完整数值平衡。
- 最终动作帧与全部音效。

当前最重要的是让观众理解：旧神残骸如何改变一个地方，玩家如何用「无音」收纳「万音」，以及一个英雄如何在饥饿的制度里变成新的收割者。
