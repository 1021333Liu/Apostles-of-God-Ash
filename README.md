# 神烬使徒 / Apostles of God-Ash

旧神已死，世界只剩回响。

《神烬使徒 / Apostles of God-Ash》是一款叙事驱动的暗黑奇幻角色扮演游戏原型。当前确定版聚焦第一章「低语田野」：玩家扮演「无音回响」，进入被旧神胃囊污染的田野，用骰子判定完成遭遇、对话、战斗与归档。

它不是传统卡牌游戏。现在的核心是：

```text
概念片演出
-> 收藏家与无声圣匣
-> 进入低语田野
-> 遭遇敌人并听见他们的证词
-> 用 D20 / D3 / D6 完成骰子战斗
-> 归档日志与掉落物
-> 面对谷仓王
-> 带着神的胃囊继续活下去
```

## 一句话

这是一个关于旧神、饥饿、人格、残骸、三位研究生和一个代码劳动力试图按时做出游戏的项目。

如果你在低语田野里听见土地说话，别回头。

大概率是 Boss 机制，也可能是策划又加需求了。

## 世界观

宇宙像一件巨大的乐器，持续释放某种频率。旧神曾经用自己的身体过滤这种频率，从中获得神力，再把神力转化为统治世界的神权。

后来宇宙产生了自我意识。理性出现的第一刻没有立刻带来秩序，反而让原本稳定的频率崩乱。旧神承载不住变化的万音，身体爆裂，只剩器官、残骸和执念散落人间。

低语田野就是被旧神胃囊影响后的地方。胃代表贪婪、食欲和吞噬，所以这片田地不再生产粮食，而开始吞噬人。

谷仓王曾经是人类英雄。他打开低语田野的粮仓，把自己献给田野，想换取粮食救下百姓。可饥饿没有停止。田野学会了通过他继续索要，于是英雄变成了新的收割者。

玩家要终止田野的吞噬，也要承受神的胃囊。问题随之出现：当一个人把旧神残骸吸收到身体里，他还是原来的自己吗？

## 游戏类型

当前确定版定位为：

```text
叙事 RPG + 骰子判定战斗 + 概念片式关卡演出
```

它更接近桌面 RPG 的战斗思路：玩家每回合选择行动，系统用骰子决定这次行动是否成立、伤害是多少、是否反弹、是否触发大招。

现在不再主打“卡牌”。行动以技能栏和骰子舞台呈现，重点是判定感、归档感和故事推进。

## 核心战斗

玩家行动轮：

- 攻击：投 D20 命中，命中后投 D3 造成伤害。
- 重击：需要比敌方防御高 5 点，成功后造成双倍伤害。
- 蓄防：本回合不攻击，下一次被攻击时防御 D20 投两次取最高，并可反弹。
- 大招：攻击累计后可用，投 D6，无视防御造成直伤。

敌方行动轮：

- 敌人自动攻击。
- 玩家自动投防御骰。
- 如果蓄防成功，防御更稳，并可能把伤害反弹回去。

一句话理解：你不是点按钮打怪，而是在向圣匣提交一份证词。骰子决定这份证词能不能穿过对方的抵抗。

4. 空腹者遭遇。
5. 农夫遭遇。
6. 谷仓王场景入场。
7. 谷仓王遭遇与战斗。
8. 失败或胜利结算演出。

- 角色立绘、背景、骰子、技能图标、掉落物图标。
- 圣匣日志和奖励归档。
