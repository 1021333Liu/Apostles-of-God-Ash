# 神烬使徒 / Apostles of God-Ash

一款正在从黑暗奇幻动作肉鸽，转向叙事卡牌骰子决斗的 Godot 游戏。  
也是一个关于“如果你的人格是靠捡别人神格碎片拼出来的，那你还算不算你自己”的严肃问题。  
当然，玩家目前最直接的问题已经从“别被麦子咬死”，变成了“为什么这个农夫防御骰又掷出了 20”。

## 2026-06-19 玩法转向说明

我们做过一版实时动作房间战斗原型，也确实能跑：玩家拿短刃进低语田野，砍空腹者，躲麦穗危险区，最后打谷仓王。

但试玩之后，我们遇到一个比较诚实的问题：**它能玩，但还不够像《神烬使徒》自己的东西。**

旧版本的问题主要有三个：

- 世界观和战斗结合不够深。文案很好，但很多时候只是贴在战斗外面。
- 实时动作部分辨识度不足。近战、危险区、清房节奏容易变成普通动作 Demo。
- 日志碎片系统没有成为主循环。玩家打完怪以后，并没有强烈感觉自己是在“理解、吞咽和归档他人的故事”。

所以项目现在转向：

```text
进入场景
-> 和中央 NPC 对话失败
-> 进入卡牌化行动选择
-> 攻击 / 防御触发骰子对抗
-> 胜利后敌人才讲出真正故事
-> 获得日志碎片和三选一奖励
```

第一场样板是“饥民农夫”：他战前只会咕噜说“饿了”“我的孩子”“天亮就下田”。打赢之后，他才会讲出自己为什么一直照常下田。

## 这是什么游戏

《神烬使徒》发生在一个旧神已经碎成世界观材料包的地方。

旧神死后，祂们的意志、器官、恐惧和执念变成了“神之残骸”。世界被一种叫“万音”的异常噪声污染，人类为了不被彻底吞掉，只好主动选择一种较轻的疯狂，把它包装成信仰、制度和城市管理方案。

于是，十座共鸣城邦诞生了。每座城邦都有一位共鸣使徒。  
他们不是普通 Boss。更准确地说，他们是“曾经试图解决问题，后来自己变成问题”的人。

玩家扮演“无韵回响”：一个没有固定频率、不会被共鸣瘟疫同化的容器。你进入城邦、击败使徒、收集神之残骸，再把这些东西装进自己身体里。

听起来很强。  
副作用是，你会逐渐变成一份很难解释的自我拼盘。

## 当前垂直切片：低语田野

第一阶段不做十座城邦。我们先做一座：**低语田野**。

这里的核心回响是暴食。  
关键词包括：

- 血肉麦穗
- 牙齿果实
- 肠道灌溉渠
- 饥饿稻草人
- 会咀嚼的谷仓
- 以及一位很努力但明显努力过头的 Boss：谷仓王

谷仓王曾经想“吃掉饥荒”，让人民不再挨饿。  
后来他忘记了“饱”是什么，只记得“继续吃”。

## 现在能玩到什么

当前仓库里仍保留旧实时动作原型，它已经有一个可运行的基础闭环：

1. 从无声圣匣出发。
2. 进入低语田野。
3. 用锈银短刃近战清怪。
4. 体验神之胃囊的击杀回血、受击闭合、满血溢出转攻击。
5. 挑战谷仓王。
6. 击败后获得神之胃囊记忆。

操作：

| 操作 | 按键 |
| --- | --- |
| 移动 | WASD / 方向键 |
| 近战 | J / Space |
| 交互/进入下一房间 | E / Enter |

但接下来的主方向不是继续扩写这套动作战斗，而是新增独立的卡牌骰子 Demo：

- 玩家和敌人左右对峙。
- 玩家每回合固定在“攻击 / 防御”中二选一。
- 双方根据行动掷 `0-20` 命中 / 防御骰，以及 `0-3` 效果骰。
- 防御大成功、攻击大成功、防御反弹都会成为明确的战斗事件。
- 胜利后进入故事、日志碎片和奖励三选一。

## 开发状态

当前版本处在一次大转向中：旧动作版能跑，但新卡牌骰子 Demo 还在搭建。

现阶段重点：

- 做出第一场“饥民农夫”卡牌骰子战。
- 把攻击 / 防御 / 骰子 / 反弹 / 大成功做成清楚的 UI。
- 让胜利后的故事和日志碎片成为核心奖励。
- 给画师明确的对峙动画组、卡牌、骰子、日志 UI 需求。
- 保留旧动作原型作为素材、世界观和反馈参考，不再把它当主玩法方向。

暂时不做：

- 十城邦全量。
- 复杂随机地图。
- 多武器库。
- 商店系统。
- 永久养成树。
- 多结局。

这些以后都会很诱人，但现在先把第一块麦田种明白。

## 更新记录

每次做了有意义的改动，都要写进 `CHANGELOG.md`。

代码负责证明“改了哪里”。  
更新日志负责解释“为什么这么改”。  
否则三个人协作两周后，项目会从黑暗奇幻变成考古现场。
