# 神烬使徒 / Apostles of God-Ash

旧神已死，世界只剩回响。

《神烬使徒 / Apostles of God-Ash》是一款叙事驱动的暗黑奇幻角色扮演游戏原型。当前确定版聚焦第一章「低语田野」：玩家扮演「无音回响」，进入被旧神胃囊污染的田野，用骰子判定完成遭遇、对话、战斗与归档。

它不是传统卡牌游戏。现在的核心是：

```text
概念片演出
-> 收藏家与无声圣匣
-> 进入低语田野
-> 遭遇敌人并听见他们的证词
-> 用 D20 / D3 / D6 完成骰子战斗
-> 归档日志与掉落物
-> 面对谷仓王
-> 带着神的胃囊继续活下去
```

## 如何开始游玩

### Windows 临时包

1. 保持 `Apostles-of-God-Ash.exe` 与 `Apostles-of-God-Ash.pck` 位于同一文件夹，并保留相同的主文件名。
2. 双击 `Apostles-of-God-Ash.exe` 启动，不需要安装 Godot。
3. 首次体验选择“新游戏”；已有章节存档时可选择“继续游戏”。

### macOS 临时包

1. 解压 `Apostles-of-God-Ash-macOS-universal.zip`。
2. 打开其中的 `Apostles of God-Ash.app`，不需要安装 Godot。
3. 当前是未签名、未公证的内部测试包；正式公开发行前仍需完成 Apple 签名与公证。

本地临时包位于 `export/`，该目录不会提交到 Git。GitHub `main` 保存游戏源码；收到测试包时，应完整保留 Windows 文件夹，或直接分发 macOS ZIP。

### 基本玩法

- 探索：使用 WASD / 方向键移动，靠近提示目标后按 Enter / Space 对话。
- 战斗：根据敌人公开的“本轮 / 下轮意图”，选择攻击、重击、蓄防或大招，再由骰子完成判定。
- 日志：按 P 打开圣匣日志，查看归档证词与掉落物。
- 暂停：按 Esc 打开暂停菜单；过场中可按 Space / Enter 跳过。
- 手柄：左摇杆 / 十字键移动，A 确认，B 返回，Y 打开日志，Start 暂停。

## 世界观

宇宙像一件巨大的乐器，持续释放某种频率。旧神曾经用自己的身体过滤这种频率，从中获得神力，再把神力转化为统治世界的神权。

后来宇宙产生了自我意识。理性出现的第一刻没有立刻带来秩序，反而让原本稳定的频率崩乱。旧神承载不住变化的万音，身体爆裂，只剩器官、残骸和执念散落人间。

低语田野就是被旧神胃囊影响后的地方。胃代表贪婪、食欲和吞噬，所以这片田地不再生产粮食，而开始吞噬人。

谷仓王曾经是人类英雄。他打开低语田野的粮仓，把自己献给田野，想换取粮食救下百姓。可饥饿没有停止。田野学会了通过他继续索要，于是英雄变成了新的收割者。

玩家要终止田野的吞噬，也要承受神的胃囊。问题随之出现：当一个人把旧神残骸吸收到身体里，他还是原来的自己吗？

## 游戏类型

当前确定版定位为：

```text
叙事 RPG + 骰子判定战斗 + 概念片式关卡演出
```

它更接近桌面 RPG 的战斗思路：玩家每回合选择行动，系统用骰子决定这次行动是否成立、伤害是多少、是否反弹、是否触发大招。

现在不再主打“卡牌”。行动以技能栏和骰子舞台呈现，重点是判定感、归档感和故事推进。

## 产品外壳与操作

游戏启动后先进入标题页，不再直接播放开场或进入战斗。当前产品壳包括：

- 继续游戏、新游戏、设置和退出游戏。
- 游戏内暂停、恢复游戏、返回标题和退出确认。
- 主音量、音效音量与全屏设置，保存到 `user://settings.cfg`。
- 版本化章节存档、临时文件和上一份备份；主存档损坏时会尝试从备份恢复。

“继续游戏”恢复的是最近一次**遭遇入口检查点**，不会恢复到骰子动画、敌方自动回合或半段过场。第一章完成后存档会标记为已完成，标题页改为“重新开始章节”。

键鼠：

- 方向键 / WASD：移动或切换选项。
- Enter / Space：确认、对话或选择行动。
- P：打开圣匣日志。
- Esc：暂停、返回或取消确认。

手柄：

- 左摇杆 / 十字键：移动或切换选项。
- A：确认；B：返回；Y：圣匣日志；Start：暂停。
- 过场中 A、B 或 Start 均可跳过。

## 核心战斗

玩家行动轮：

- 攻击：投 D20 命中，命中后投 D3 造成伤害。
- 重击：需要比敌方防御高 5 点，成功后造成双倍伤害。
- 蓄防：本回合不攻击，下一次被攻击时防御 D20 投两次取最高，并可反弹。
- 大招：攻击累计后可用，投 D6，无视防御造成直伤。

敌方行动轮：

- 敌人按公开序列行动，界面会同时显示本轮与下轮意图。
- 攻击意图：敌人自动攻击，玩家自动投防御骰；蓄防会改为 2D20 取高，并可能反弹。
- 防御意图：玩家普通攻击要面对敌方 2D20 取高，重击可破除额外防御骰；随后敌人不会攻击。

一句话理解：你不是点按钮打怪，而是在向圣匣提交一份证词。骰子决定这份证词能不能穿过对方的抵抗。

## 音频状态

当前版本会在启动时生成并缓存一组原创程序化短音效，用于菜单焦点与确认、敌方意图、骰子滚动与落点、攻击/蓄防/重击/大招、命中/格挡/落空、田野脚步、日志归档、奖励与胜负反馈。它们不依赖来源不明的第三方音频，适合作为当前垂直切片的即时操作反馈，但不冒充最终商业音频资产。

演出视频继续使用各自 `.ogv` 文件中的内嵌音轨；播放过场时不会额外叠加通用程序音效。

正式配乐、低语田野区域氛围、不同地面材质脚步、角色与敌人专属攻击/受击拟音，以及过场最终混音与响度统一，仍需要后续正式音频制作。
