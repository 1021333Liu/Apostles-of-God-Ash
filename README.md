# 神烬使徒 Godot

《神烬使徒》Godot 独立项目工作区。

## 项目方向

- 类型：黑暗奇幻动作肉鸽。
- 核心体验：玩家作为“无韵回响”，进入共鸣城邦，击败使徒，收集神之残骸，并用残骸拼装自己的战斗风格与人格倾向。
- 第一阶段目标：只做“无声圣匣 + 低语田野 + 谷仓王 Boss”的垂直切片。

## 当前文件

- `docs/神烬使徒_项目世界观与垂直切片规划.md`：项目世界观、十座城邦、第一阶段垂直切片、组员分工与验收标准。
- `docs/DEMO_VERTICAL_SLICE_SPEC.md`：低语田野 Demo 的可执行规格表。
- `docs/planning/00_PRODUCTION_INDEX.md`：垂直切片制作索引，按美术、地图关卡、文案、道具、数值和任务拆分。
- `docs/planning/07_TEAM_GIT_WORKFLOW.md`：三人团队 Git 分支、文件边界和文案交付流程。
- `project.godot`：Godot 4.x 工程配置。
- `scenes/main.tscn`：当前可运行主场景。
- `scripts/main.gd`：当前垂直切片原型逻辑。
- `assets/`：后续放 SVG、贴图、音频等资源。

## 第一轮建议

1. 打开 Godot 工程并运行 `scenes/main.tscn`。
2. 验证无声圣匣进入低语田野的固定房间链。
3. 调整主角近战手感、敌人速度和谷仓王阶段节奏。
4. 确认神之胃囊的击杀回血、受击闭合、满血溢出转攻击是否成立。
5. 等核心循环成立后，再拆分玩家、敌人、Boss、HUD 为独立场景。
