# 《神烬使徒》低语田野 Demo 执行规格

## Demo 目标

第一版只验证一个商业可卖点：玩家不是在刷普通装备，而是在用神之残骸拼装一个越来越危险的自我。

本 Demo 的目标体验是 5-8 分钟内完成一轮：

1. 从无声圣匣出发。
2. 进入低语田野的固定房间链。
3. 用锈银短刃近战清怪。
4. 体验神之胃囊的击杀回血与饥饿惩罚。
5. 击败谷仓王。
6. 回到圣匣并获得一段记忆。

## 房间链路

| 顺序 | 房间 | 功能 | 验证点 |
| --- | --- | --- | --- |
| 0 | 无声圣匣 | 起点、复活、收藏家提示 | 死亡循环是否成立 |
| 1 | 低语田野入口 | 2 个空腹者 | 基础移动、近战、击杀回血 |
| 2 | 血肉麦田 | 空腹者 + 饥民农夫 + 咬人麦穗 | 近战走位和地面威胁 |
| 3 | 肠道灌溉渠 | 饥民农夫 + 消化液区域 | 移动限制和远程投掷压力 |
| 4 | 饥饿谷仓 | 饥饿稻草人 + 空腹者 | 精英召唤与爆发窗口 |
| 5 | 谷仓王胃室 | Boss 三阶段 | Boss 悲剧感、阶段变化、残骸奖励 |

## 玩家

| 项 | 首版参数 |
| --- | --- |
| 生命 | 100 |
| 移动 | WASD / 方向键 |
| 攻击 | J / Space |
| 交互 | E / Enter |
| 武器 | 锈银短刃 |
| 攻击节奏 | 三连击，第三击更高伤害 |

## 神之胃囊

| 层级 | 设计 |
| --- | --- |
| 机制 | 击杀回血；满血溢出转成短时攻击加成 |
| 代价 | 受击后胃囊闭合，短时间无法通过击杀回血 |
| 叙事 | 谷仓王曾把胃献给土地，希望吃掉饥荒 |
| 侵蚀 | 主角胸口出现暗红胃纹 |

## 敌人

| 敌人 | 行为 | 作用 |
| --- | --- | --- |
| 空腹者 | 追踪近战，低血加速 | 训练走位和基础击杀 |
| 饥民农夫 | 保持距离，投掷牙齿作物 | 迫使玩家主动接近 |
| 饥饿稻草人 | 召唤麦浪，缓慢压迫 | 精英房节奏峰值 |
| 谷仓王 | 召唤、消化液、胃囊暴露 | 垂直切片终局验证 |

## Boss 阶段

| 阶段 | 触发 | 行为 |
| --- | --- | --- |
| 一 | 100%-67% HP | 追踪玩家，召唤空腹者 |
| 二 | 67%-34% HP | 生成消化液，限制走位 |
| 三 | 34%-0% HP | 攻击更密，周期性暴露胃囊弱点 |

## 当前实现入口

- Godot 主场景：`res://scenes/main.tscn`
- 主逻辑脚本：`res://scripts/main.gd`
- 工程配置：`res://project.godot`

当前版本使用程序化绘制来验证玩法节奏。美术资产进入项目前，先不要拆出复杂 prefab；等近战、残骸、Boss 节奏成立后，再把玩家、敌人、房间和 HUD 拆成独立场景。
