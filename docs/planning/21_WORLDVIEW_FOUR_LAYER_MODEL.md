# 《神烬使徒》底层世界观四层模型

## 文档用途

本文档用于统一《神烬使徒 / Apostles of God-Ash》的底层世界观逻辑。

它不是宣传文案，也不是完整百科，而是给后续美术、文案、关卡、日志系统和战斗设计使用的世界观骨架。

参考的拆解方式是“四层世界观模型”：

1. 物理 / 自然法则
2. 历史与势力
3. 文化与信仰
4. 日常质感

玩家真正摸到的是第 4 层，但第 4 层的可信度来自下面三层。

## 一句话核心

《神烬使徒》的世界不是“神还在统治人”，而是“神死后留下的器官、规则和欲望还在运行”。

人类为了活下去，把这些残骸做成制度、宗教、技术和日常生活。玩家扮演的“无韵回响”进入这些已经失控的区域，击败使徒，把旧神残骸装进自己身体里，获得力量，也被它改写。

## 总结构

```text
第 4 层：日常质感
空碗、名单、谷仓、血麦、胃囊 HUD、日志碎片、死亡归档

第 3 层：文化与信仰
丰收节、献祭登记、谷仓王崇拜、饥饿伦理、残骸崇拜

第 2 层：历史与势力
旧神陨落、残骸争夺、城邦使徒、收藏家、低语田野饥荒史

第 1 层：物理 / 自然法则
神死后残骸仍在运行；每个残骸携带一种欲望和代价
```

## 第 1 层：物理 / 自然法则

### 核心规则：神烬法则

旧神已经死亡，但神的残骸没有真正停止。

胃、眼、骨、舌、血、心脏、影子等残骸仍然携带规则。它们不再拥有完整意志，却会继续执行旧神死亡前残留的本能。

这个世界的基本规律是：

> 力量不是免费魔法，而是残骸的本能。  
> 每个神之残骸都在继续执行它死亡前最后一种欲望。

因此，玩家获得能力不是“装备技能”，而是“把一个神的坏习惯装进身体”。

### 残骸示例

| 神之残骸 | 底层规则 | 可能代价 |
| --- | --- | --- |
| 神之胃囊 | 把饥饿变成物理法则，吞噬生命并转化为恢复、力量和记忆 | 永远不满足，区分不了进食与献祭 |
| 神之眼 | 被看见的东西更接近真实，可用于审判、预言、监控 | 无法遗忘，被观看者失去隐私和谎言 |
| 神之骨 | 让死亡后的形体继续站立，可支撑军队、建筑和圣物 | 死者难以真正安息 |
| 神之舌 | 誓言、谎言、命令具备强制力 | 语言变成枷锁，制度被写进身体 |
| 神之血 | 让伤口、契约、污染流动和传播 | 个体边界被溶解，罪和力量会传染 |

### 对玩法的要求

- 每个残骸能力都必须有收益和代价。
- 残骸不是普通装备，而是人格拼装材料。
- 玩家越强，越不像一个完整的人。
- 系统反馈要让玩家意识到自己在“使用神的残余欲望”。

## 第 2 层：历史与势力

### 世界历史：旧神陨落后的残骸时代

旧神陨落后，世界没有迎来自由，反而进入残骸时代。

神的尸体碎片散落在不同地区。城邦、教团、军队、商会、收藏家和普通居民都开始争夺、研究、崇拜或依赖这些残骸。

旧神死后，世界真正的问题变成：

> 谁有资格使用神的尸体？  
> 谁来承担使用它的代价？  
> 当代价开始有效时，谁还愿意停止？

### 主要势力

| 势力 | 世界观功能 | 设计作用 |
| --- | --- | --- |
| 圣匣收藏者 | 回收、保存、实验神之残骸 | 玩家复活、归档、能力装配的入口 |
| 城邦使徒 | 获得残骸后被区域制度改造的人或怪物 | 各区域 Boss 与核心残骸来源 |
| 残骸教会 | 相信旧神没有死，只是被拆散 | 宗教冲突、禁忌、仪式来源 |
| 技术工坊 | 把残骸当能源、武器、医疗器官使用 | 科技与神性混合的装备来源 |
| 普通居民 | 依赖残骸带来的粮食、安全和秩序 | 让世界不是简单的受害者叙事 |

### 低语田野的历史案例

低语田野不是普通怪物巢穴，而是一段失败的社会史。

1. 饥荒爆发。
2. 谷仓王打开粮仓救人。
3. 粮食仍然不够。
4. 村民开始接受献祭。
5. 神之胃囊让献祭真的变成粮食。
6. 制度因为“有效”而被保留下来。
7. 谷仓王献出自己的胃，试图让献祭到他为止。
8. 田地接受了他，也学会了通过他继续索要。
9. 饥荒没有被解决，只是学会了用谷仓王的嘴说话。

谷仓王不是单纯的坏人。他是一个曾经解决问题的人，后来被解决方案吞掉。

## 第 3 层：文化与信仰

### 核心逻辑

人类不会把残骸直接称为“怪物器官”。为了继续生活，他们会给残骸包装成信仰、伦理、习俗和制度。

这层解释的是：

> 为什么错误的事能够长期存在？  
> 因为人们发明了一套语言，让自己可以继续做下去。

### 低语田野的文化信仰

| 文化现象 | 表层说法 | 底层含义 |
| --- | --- | --- |
| 丰收节 | 感谢谷仓王与田地 | 献祭成功后的集体自我安慰 |
| 名单登记 | 公平分配粮食 | 决定谁可以被牺牲 |
| 空碗仪式 | 每户领取粮食的凭证 | 饥饿被制度化后的标记 |
| 谷仓王崇拜 | 他让大家活过冬天 | 用英雄叙事遮盖后续罪责 |
| 田地低语 | 土地在回应祈祷 | 神之胃囊的饥饿在模仿民意 |
| 静默者 | 不谈旧事的人 | 参与者用沉默维持秩序 |

### 文案基调

低语田野的文案不能只写“邪恶田地”和“可怕怪物”。

它应该反复强调：

- 村民不是完全无辜。
- 谷仓王不是一开始就邪恶。
- 献祭不是突然出现的疯狂，而是被饥荒、秩序和有效性一步步合理化。
- 最可怕的不是吃人，而是吃人之后形成了流程。

### 对敌人设计的要求

| 敌人 | 信仰 / 制度含义 |
| --- | --- |
| 空腹者 | 被排除在分配之外的人，最后连身份也被吃掉 |
| 饥民农夫 | 把杀人当农活的人，制度执行者 |
| 饥饿稻草人 | 恐惧和命令被自动化后的守卫 |
| 谷仓王 | 救主、罪人、制度借口和神之胃囊容器 |

## 第 4 层：日常质感

### 玩家实际摸到的世界

玩家不会直接读百科，而是通过食物、衣服、建筑、街景、俚语、UI 和战斗反馈感受到前三层。

低语田野的日常质感应该来自这些东西：

- 路牌正面写“丰收”，背面刻名单。
- 每户门口挂着空碗，代表分粮资格。
- 农夫仍然准时下田，哪怕田里种的是人。
- 稻草人穿着偷粮者的衣服。
- 谷仓里没有粮声，只有胃液和记账声。
- 小怪说的不是“我要杀你”，而是“别插队”“名单别乱”“今年不能荒”。
- HUD 不是普通血条，而是“器官状态”和“样本归档”。
- 日志碎片不是收藏品，而像被胃液泡过的证词。

### 日常物件清单

| 物件 | 用途 | 世界观含义 |
| --- | --- | --- |
| 空碗 | 环境、日志、UI 图标 | 等待分粮，也等待献祭 |
| 名单 | 房间提示、日志碎片、敌人台词 | 分配秩序，也是牺牲秩序 |
| 谷仓钥匙 | Boss 记忆、通关奖励 | 救人的权力，后来变成杀人的权力 |
| 血麦 | 危险地面、房间背景 | 献祭获得成果后的证据 |
| 胃液渠 | 地形阻碍、伤害区域 | 饥饿被工程化 |
| 破衣稻草人 | 精英敌人视觉 | 偷粮者和旧命令的残留 |
| 胃囊纹 | 玩家能力反馈 | 玩家正在被残骸改写 |
| 银白晶片 | 日志碎片 | 真相被采样、切片、归档 |

## 日志系统在四层模型中的位置

日志系统是玩家从第 4 层向下挖掘的工具。

玩家一开始只看到：

- 怪物
- 麦田
- 谷仓
- 胃囊回血
- 采样记录

通过日志碎片，玩家逐步发现：

1. 空腹者不是怪，是排队排到最后的人。
2. 农夫不是疯子，是制度执行者。
3. 稻草人不是守卫，是旧命令自动化。
4. 村民不是纯受害者，他们参与过献祭。
5. 谷仓王不是单纯 Boss，他是第一个想停止灾难但失败的人。
6. 神之胃囊不是奖励，而是饥荒本身的残骸。

因此，日志系统不是附属内容，而是本作叙事成立的关键系统。

## 对玩法设计的直接要求

### 战斗

- 击杀不是清杂兵，而是采样世界。
- 每个敌人死亡后都应有机会掉出一段“人曾经存在过”的证据。
- 胃囊回血必须带有代价感，不能像普通吸血词条。
- Boss 弱点窗口应像仪式，而不是普通破防。

### 关卡

- 每个房间应体现一条制度链：
  - 入口：登记和名单。
  - 血肉麦田：成果和代价。
  - 肠道灌溉渠：献祭被工程化。
  - 饥饿谷仓：储存、分配、空缺。
  - 谷仓王胃室：救主被制度吞掉。

### 美术

- 不做普通农场恐怖。
- 重点是“农业、制度、饥饿、神性残骸”的混合。
- 背景要提供日常证据：碗、名单、农具、登记牌、分粮痕迹。
- 角色必须能看出“曾经是人”，而不是纯怪物。

### UI

- UI 应像圣匣档案、医学记录和器官标本。
- 生命条、胃囊状态、日志碎片、死亡回收都要服务“归档样本”的感觉。
- 圣匣不是菜单，而是玩家被保存、拆解、重组的地方。

### 文案

- 少写解释，多写证据。
- 少写邪恶宣言，多写流程语言。
- 小怪台词要像日常抱怨、制度口号或残留记忆。
- Boss 台词要体现“我曾经想让事情停下”。

## 低语田野的第一故事

日志系统第一阶段要拼出的核心问题是：

> 低语田野为什么开始吃人？

建议第一故事梗概：

低语田野原本是饥荒中的村庄。谷仓王起初打开粮仓救人，后来粮食不够，村民开始接受献祭。献祭一开始针对外乡人、罪犯和“可以被牺牲的人”，后来扩大到每一户。谷仓王献出自己的胃，试图替所有人承受饥饿。血肉麦穗真的长出来后，制度获得了成果，于是没人再愿意停止。最后，饥荒没有被解决，只是学会了用谷仓王的嘴说话。

## 结论

《神烬使徒》的表层是动作肉鸽，底层是玩家通过战斗挖掘一个制度如何把人吃掉。

低语田野不是“打败怪物拯救村庄”的故事，而是：

> 玩家进入一套已经成功运行过的错误制度，  
> 击败它的执行者，  
> 收集它留下的证词，  
> 最后把它的神之残骸装进自己身体里。

这就是本作最重要的世界观张力：

> 你不是在净化世界。  
> 你是在用世界的病灶拼出自己。
