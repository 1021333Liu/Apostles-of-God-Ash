# 道具、残骸与奖励设计

## 系统定位

《神烬使徒》的道具不应只是数值装备。核心分类应该是：

- 武器：改变玩家攻击方式。
- 残骸：改变战斗循环和人格倾向。
- 晶片：提供叙事、解锁和真相。
- 临时回响：只在本轮有效的小型 Build 选择。

首版 Demo 只实现锈银短刃、神之胃囊、静默者晶片和少量临时回响概念。

## 武器：锈银短刃

| 项 | 设计 |
| --- | --- |
| 定位 | 初始近战武器 |
| 攻击 | 三连击 |
| 第 1 击 | 快速横斩，基础伤害 |
| 第 2 击 | 同基础伤害，用于稳定连段 |
| 第 3 击 | 更高伤害，轻微击退 |
| 叙事 | 由神血银锻成，能切割回响残留 |

后续扩展：

- 短刃不是永久唯一武器，但首版不要加武器库。
- 可以先把短刃手感打磨到及格，再考虑斧、镰、铃等武器。

## 残骸：神之胃囊

| 层级 | 内容 |
| --- | --- |
| 槽位 | 神之心或特殊器官槽 |
| 核心收益 | 击杀敌人回复生命 |
| 溢出收益 | 满血时治疗溢出转化为短时攻击力 |
| 核心代价 | 受击后进入胃囊闭合，短时间无法击杀回血 |
| 侵蚀表现 | 胸口胃纹加深，收藏家台词变化 |
| 叙事真相 | 谷仓王曾献出胃，希望替人民承受饥荒 |

首版数值：

| 参数 | 建议 |
| --- | --- |
| 普通击杀回血 | 8 |
| Boss 击杀回血 | 10 |
| 溢血攻击加成 | +12 |
| 溢血持续 | 4 秒 |
| 受击闭合 | 2.2 秒 |

设计风险：

- 如果回血太强，玩家会无脑换血。
- 如果闭合太长，玩家会觉得残骸不稳定但不好玩。
- 如果溢出反馈不明显，玩家无法理解“过量治疗转攻击”。

## 晶片：静默者晶片

| 项 | 设计 |
| --- | --- |
| 定位 | 叙事收集物 |
| 来源 | 精英房、Boss 后、隐藏事件 |
| 首版功能 | 解锁一段低语田野真相文本 |
| 长期功能 | 推进结局分歧、揭露收藏家旧容器记录 |

首版晶片主题：

- 村民主动参与献祭。
- 谷仓王曾经被感谢。
- 祭台比天气可靠。
- 玩家吸收胃囊后也会面对同类问题。

## 临时回响候选

首版可先不实现完整三选一，但策划上应预留。

| 名称 | 效果 | 代价 | 适配房间 |
| --- | --- | --- | --- |
| 余粮 | 进入新房间时回复 6 HP | 胃囊闭合时间 +0.3 秒 | 入口、麦田 |
| 饥民步 | HP 低于 35% 时移速 +18% | 受到地面伤害 +10% | 灌溉渠 |
| 牙种 | 第三击在命中点生成短暂伤害区 | 攻击冷却 +0.05 秒 | 麦田 |
| 空仓 | 满血溢出攻击加成提高 | 最大生命 -10 | Boss 前 |
| 银雾嚼咽 | 击杀后短暂减伤 | 记忆晶片掉率降低 | 精英房 |

## 奖励节奏

| 节点 | 奖励 |
| --- | --- |
| 入口房完成 | 第一次胃囊反馈 |
| 血肉麦田完成 | 晶片或临时回响 |
| 肠道灌溉渠完成 | 临时回响 |
| 饥饿谷仓完成 | 静默者晶片 |
| 谷仓王死亡 | 神之胃囊归档 + 关键记忆 |

## 长期残骸规则

每个残骸都必须包含：

1. 一个明确战斗循环。
2. 一个能被玩家感知的代价。
3. 一段使徒或旧神记忆。
4. 一个侵蚀表现。
5. 一个结局倾向。

如果某个道具只有加伤害、加血量、加速度，它不应该被称为残骸，只能作为临时回响或普通强化。

## 验收标准

- 神之胃囊能明显改变玩家打法。
- 玩家能理解受击后为什么不能回血。
- 溢血攻击反馈足够明显。
- 晶片文本让玩家重新理解谷仓王，而不是只补设定。
- 后续道具扩展不破坏“残骸=人格碎片”的核心定位。
