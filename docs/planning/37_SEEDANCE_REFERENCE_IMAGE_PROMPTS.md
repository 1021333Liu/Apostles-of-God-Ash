# 《神烬使徒》Seedance 参考图提示词批次

版本：2026-06-30  
用途：把 36 号叙事脚本改写成“可喂参考图”的 Seedance / 即梦视频模型提示词。  
方法来源：参考 `qian-short-drama-prompter` 的工作流，先做导演推理，再写单 clip；每条 prompt 明确参考图、镜头起点、运动路径、落幅、光源、主体动作、物件动作和声音。  
边界：本文件只整理提示词，不代表视频已生成。

## 01 Qian 方法在本片里的用法

- 先定每个 clip 的戏剧功能，再写镜头，不把剧情摘要塞给模型。
- 每个 clip 只服务一个主目标：醒来、田野揭示、敌人证据、Boss 悲剧、倒地回收、收藏家追问。
- 参考图只锁定角色外观、场景材质和道具形状；不要让模型生成文字、字幕、UI 文案。
- 普通反应少用复杂运镜，用干净切镜、固定近景、短推近；只有揭示、倒地、归档、追问使用明确运动。
- 对话类镜头若要口型，必须写“从第一帧明确开口说话、嘴部清晰可见”；本概念片多数旁白后期配，不要求模型生成口型。
- 灯光必须有来源：烛光、圣匣冷光、灰天漫反射、谷仓门缝、骰子边缘冷光。
