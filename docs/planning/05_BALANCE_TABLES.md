# 首版数值表

## 数值目标

首版数值不追求精确平衡，目标是建立可调参基线。所有参数应该让测试者能回答：

- 近战是否爽快？
- 被打是否清楚？
- 胃囊回血是否有诱惑？
- 地面危险是否制造走位，而不是纯惩罚？
- Boss 是否有阶段感？

## 优先调参顺序

1. 攻击冷却和攻击范围。
2. 空腹者 HP 和速度。
3. 受击闭合时间。
4. 地面危险半径和伤害。
5. Boss HP。
6. Boss 特殊行为频率。

不要一轮同时改太多参数。每次只改 1-3 项，否则无法判断变化来源。

## 验收标准

- 新手测试者首次死亡后愿意再来一局。
- 熟悉后 5-8 分钟可以完成一轮。
- 玩家能说出自己为什么死。
- 胃囊让玩家想击杀回血，但不敢无脑换血。
- Boss 三阶段在不看文档时也能感知。
