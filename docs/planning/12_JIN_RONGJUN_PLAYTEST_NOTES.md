# 金荣俊试玩记录：关卡 / HUD / 表现

日期：2026-06-01

## 无声圣匣

- HUD 信息层级清楚：房间名、HP、胃囊状态、目标和交互提示都容易理解。
- HUD 视觉简陋，缺少《神烬使徒》的暗黑奇幻质感。

### 对应任务

- 用暗红生命条、银色边框、生铁黑褐底色改善占位 HUD。
- 为胃囊状态增加图标或更明显的状态块。
- 保持当前信息层级，不要为了装饰降低扫读效率。

## 低语田野入口

- 玩家与敌人容易区分。
- 攻击弧线和命中反馈不明显。
- 当前弧线太短；角色仍为小型圆球时，玩家难以判断近战距离。
- 希望适度增加近战攻击距离。
- 击杀回血不明显：缺少角色附近提示、动画反馈和音效提示。

### 对应任务

- 与核心玩法负责人确认后，适度增加近战攻击范围。
- 加长攻击弧线，增强方向性和三连击层次。
- 增加命中确认，例如更明显的受击闪烁、伤害数字或短促停顿。
- 增加击杀回血反馈，例如角色周围暗红或银白脉冲、醒目的 `+HP`
  文本，并预留音效接入点。

## 后续试玩待观察

- 血肉麦田：危险区是否像伤害区域，而不是装饰。
- 肠道灌溉渠：远程与地面压力是否看得懂。
- 饥饿谷仓：精英攻击前摇是否明确。
- 谷仓王胃室：阶段变化和弱点暴露是否足够醒目。

## 血肉麦田

- 从第二个战斗房开始，难度陡然提升。
- 清空敌人后如果不迅速进入下一房，玩家仍可能被危险区伤害。
- 单次危险区掉血约为最大 HP 的十分之一，容错偏低。
- 红色危险区容易识别，受伤来源也比较清楚。
- 胃囊闭合机制不容易理解。
- 玩家没有注意到黄褐色远程敌人，说明农夫的视觉识别和远程攻击前摇不足。

### 对应任务

- 清房后立即停用或清除当前房间危险区，给玩家安全的观察与喘息时间。
- 与核心玩法负责人讨论危险区伤害和触发频率；优先降低连续误踩的惩罚。
- 在 HUD 中把胃囊闭合解释为：受击后短时间内，击杀不再回血。
- 为胃囊闭合增加醒目倒计时、图标切换和角色周围反馈。
- 强化远程农夫轮廓，并为投掷行为增加可读前摇或落点提示。

## 肠道灌溉渠：阻断完整试玩

- 当前难度过高，测试者无法稳定进入第四个战斗房。
- 最难躲避的是突然出现的红色伤害地块；多个危险叠加后画面混乱。
- 减速效果不明显，玩家主要感受到的是高频扣血。
- 现阶段应先保证测试者能够体验后续关卡，再继续验证精英房和 Boss。

### 代码核对

- 饥民农夫每 `2.4` 秒在玩家附近直接生成牙齿作物危险区。
- 牙齿作物没有独立的预警阶段，生成后即可造成伤害。
- 肠道灌溉渠同时存在三个常驻消化液区域。
- 危险区每 `0.65` 秒可重复造成伤害，容易连续扣血。

### P0 调整建议

- 给远程农夫增加明显的抬手前摇：角色先进入容易识别的投掷姿态，再显示
  红色落点轮廓，延迟后才让地块开始造成伤害。玩家必须有足够时间主动躲避。
- 降低首次试玩的危险区伤害或 tick 频率，避免连续误踩快速死亡。
- 清房后清除危险区。
- 先制作开发试玩模式，允许跳房或提高容错，以便验证精英房和 Boss。
- 强化减速反馈，或在首轮测试中先降低减速影响。

## 本轮结论与任务优先级

### P0：先保证完整切片可以试玩

1. 为远程农夫投掷增加明显抬手前摇、落点预警和伤害延迟。
2. 降低动态危险区伤害或 tick 频率，避免突然连续扣血。
3. 清房后立即清除危险区，让玩家有喘息和观察时间。
4. 增加开发试玩模式，允许提高容错或跳房，以便验证精英房和 Boss。

### P1：改善核心战斗反馈

1. 与核心玩法负责人确认后，适度增加近战攻击范围。
2. 加长并强化攻击弧线，让视觉范围更接近实际判定。
3. 加强命中确认，例如受击闪烁、伤害数字或短促停顿。
4. 强化击杀回血反馈，例如角色周围脉冲、醒目的 `+HP` 文本和音效。
5. 清楚解释胃囊闭合：受击后短时间内，击杀无法回血。

### P2：改善占位 HUD 美感

1. 保留当前清楚的信息层级。
2. 使用暗红生命条、银色边框和生铁黑褐底色。
3. 为胃囊状态增加开启、闭合和溢血刃图标或状态块。

### 尚未完成验证

- 饥饿谷仓：精英攻击前摇是否明确。
- 谷仓王胃室：阶段变化和弱点暴露是否足够醒目。
- 完整切片：首次通关时长是否达到目标 `5-8` 分钟。
