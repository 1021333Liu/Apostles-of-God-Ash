# 第三方素材来源

项目只接入许可证清楚、允许修改与再分发的公开免费素材。优先使用 `CC0`。

## 已接入

### `tstamborski/pixelart-icons`

- 来源：<https://github.com/tstamborski/pixelart-icons>
- 许可证：`CC0 1.0 Universal`
- 项目内许可证副本：
  `assets/sprites/ui/LICENSE.pixelart-icons-CC0.txt`
- 已使用：
  - `ui_hp_heart.png`
  - `ui_stomach_open.png`
  - `ui_stomach_closed.png`
  - `ui_stomach_overflow.png`
  - `ui_memory_shard.png`

## 已下载但尚未接入

- 来源：<https://github.com/sparklinlabs/superpowers-asset-packs>

- 用途候选：
  - 像素字体参考。
  - 中世纪奇幻 HUD 框体参考。
  - 通用音效与反馈特效候选。

## 规则

- 不接入许可证不明素材。
- 不接入仅限非商业用途素材。
- 不复制其他商业游戏的图标、界面或角色素材。
- 即使 `CC0` 不要求署名，也保留来源和许可证副本。
