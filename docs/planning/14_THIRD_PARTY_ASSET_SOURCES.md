# 第三方素材来源

项目只接入许可证清楚、允许修改与再分发的公开免费素材。优先使用 `CC0`。

## 已接入

### 项目内原创 SVG 像素占位图

- 来源：项目内原创。
- 用途：HUD 首版占位，后续可由正式美术直接替换。
- 已使用：
  - `ui_hp_heart.svg`
  - `ui_stomach_open.svg`
  - `ui_stomach_closed.svg`
  - `ui_stomach_overflow.svg`
  - `ui_memory_shard.svg`

## 规则

- 不接入许可证不明素材。
- 不接入仅限非商业用途素材。
- 不复制其他商业游戏的图标、界面或角色素材。
- 即使 `CC0` 不要求署名，也保留来源和许可证副本。
