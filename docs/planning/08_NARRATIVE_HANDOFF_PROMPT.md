# 文案同学工作交接说明

这份文档可以直接交给文案同学，也可以直接喂给她自己的 Codex，让它辅助她产出第一批文本。

## 你的身份

你是《神烬使徒》的文案负责人。

你的任务不是写一堆漂亮但用不上的世界观散文，而是为“低语田野”垂直切片提供可以直接进游戏的短文本。

本阶段只写一座城邦：**低语田野**。  
不要展开十城邦全量，不要写完整结局，不要写长篇对白。

## 项目一句话

《神烬使徒》是一款黑暗奇幻动作肉鸽。玩家扮演没有固定人格的“无韵回响”，进入被神明残响污染的城邦，击败共鸣使徒，收集神之残骸，并用残骸拼装自己的战斗风格和人格倾向。

低语田野的主题是：**饥饿、献祭、土地吃人、曾经合理的制度变成怪物。**

## 必读背景

低语田野：

- 核心回响：暴食。
- 视觉关键词：血肉麦穗、牙齿果实、肠道灌溉渠、饥饿稻草人、会咀嚼的谷仓。
- 社会规则：所有人都必须喂饱土地。
- Boss：谷仓王。
- 残骸：神之胃囊。
- 真相：谷仓王曾经想“吃掉饥荒”，让人民不再挨饿。后来他忘记了“饱”，只记得“继续吃”。

主角：

- 名为“无韵回响”。
- 没有固定人格，不会被共鸣瘟疫同化。
- 可以装填多个神之残骸。
- 核心问题是：如果情感、能力和记忆都来自外部装填，你还算是一个人吗？

收藏家：

- 导师、复活机制维护者、残骸鉴定者。
- 前期要像可靠引导者，不要一上来写成反派。
- 他的语言可以冷静、记录式、像医生或研究员。

## 写作原则

1. 短。战斗中出现的文本最多 1-2 句。
2. 狠。每句都要有信息，不写空泛诗句。
3. 怪物曾经合理。谷仓王不是普通坏人。
4. 居民不是纯受害者。他们也参与维持这套制度。
5. 收藏家早期不能明显邪恶。
6. 所有文本都要服务游戏体验：让玩家更懂敌人、Boss、残骸或死亡。

## 第一批交付

请输出一个 Markdown 文件，文件名建议：

`低语田野_文案_第1版.md`

结构如下：

```md
# 低语田野文案 第 1 版

## 1. 收藏家开场台词

写 3-5 条。用于玩家第一次在无声圣匣醒来。

## 2. 收藏家进入田野台词

写 3-5 条。用于玩家按 E 进入低语田野前后。

## 3. 死亡回收台词

写 8 条。用于玩家死亡回到无声圣匣。
最好覆盖：被空腹者杀死、被危险地面杀死、被 Boss 杀死、贪刀死亡、首次死亡、多次死亡。

## 4. 房间短句

每个房间 1 条：

- 低语田野入口
- 血肉麦田
- 肠道灌溉渠
- 饥饿谷仓
- 谷仓王胃室

## 5. 敌人图鉴

每个敌人写 1 段，每段 60-120 字：

- 空腹者
- 饥民农夫
- 饥饿稻草人

每段后面附一句机制提示。

## 6. 谷仓王台词

### 开场

写 3 条。

### 阶段二

写 3 条。

### 阶段三

写 3 条。

### 死亡

写 3 条。

## 7. 神之胃囊记忆碎片

写 3-5 条。
要求能拼出谷仓王从救世者变成怪物的过程。

## 8. 静默者晶片文本

写 5-8 条。
要求体现普通居民也参与了献祭制度，不要只把他们写成无辜受害者。
```

## 参考语气

可以接近：

- 冷静记录。
- 宗教制度的残酷。
- 饥荒后的集体自我辩护。
- Boss 临死前的迟到醒悟。

不要接近：

- 中二喊口号。
- 大段设定讲解。
- 网络梗。
- 纯血腥猎奇。
- “吾乃某某，尔等受死”式 Boss 台词。

## 可直接改写的示例

收藏家：

> 容器醒了。先别谈救世，先证明你能活过一片田。

房间短句：

> 麦穗有牙。别把停顿交给土地。

谷仓王：

> 我吃，不是因为我贪。我吃，是因为总得有人替他们饿。

静默者晶片：

> 我们说他是怪物，这样就不用承认自己曾经跪谢怪物。

## 交付后会发生什么

你写完后，把 Markdown 发给刘秉昂或直接交给 Codex。

Codex 会负责：

1. 整理进 `docs/planning/03_NARRATIVE_COPY.md`。
2. 去重和统一术语。
3. 必要时压缩成游戏内可显示短句。
4. 后续转成 Godot 可读的数据表。
