# 金荣俊试玩检查表：关卡 / HUD / 表现

目标：试玩一次当前低语田野切片，并记录玩家是否看得懂关卡、HUD 和战斗反馈。

## 操作

| 操作 | 按键 |
| --- | --- |
| 移动 | `WASD` / 方向键 |
| 近战 | `J` / `Space` |
| 交互、进入下一房间 | `E` / `Enter` |

## 边玩边观察

### 无声圣匣

- 能否立刻看懂这里是安全起点？
- 能否立刻找到进入田野的交互提示？
- 房间名、HP、胃囊状态和目标是否容易扫读？

### 低语田野入口

- 玩家、空腹者和出口是否容易区分？
- 攻击有没有明确弧线？
- 打中敌人和击杀敌人时，反馈是否足够明显？
- 第一次击杀回血时，能否注意到胃囊机制？

### 血肉麦田

- 红色危险区是否一眼像“会造成伤害”，而不是普通装饰？
- 受伤时能否理解伤害来源？
- 受伤后胃囊闭合的提示是否足够清楚？
- 远程敌人与普通敌人是否容易区分？

### 肠道灌溉渠与饥饿谷仓

- 不同房间是否有明显视觉变化？
- 地面危险、远程投掷和精英攻击是否看得懂？
- 战斗信息是否太杂，HUD 是否挡住关键区域？

### 谷仓王胃室

- Boss 与普通敌人的体量、颜色和压迫感是否明显不同？
- 阶段变化是否足够醒目？
- 暴露胃囊弱点时，能否立刻知道应该攻击哪里？
- Boss 房是否有明显终局感？
