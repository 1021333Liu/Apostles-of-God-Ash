# 美术方向与资产清单

## 核心视觉定位

低语田野不是普通农田，而是“土地把人类饥荒学成了一种宗教”。画面应同时出现三类材质：

- 腐败农业：麦田、谷仓、农具、草垛、灌溉渠。
- 神性残骸：银白液态金属、晶片、器官形态的圣物。
- 饥饿血肉：牙齿果实、胃纹、肠道沟渠、会咀嚼的地面。

禁止把它做成普通僵尸农场或单纯血腥场景。低语田野的卖点是“饥饿被制度化”，不是只有恶心。

## 色彩基调

| 用途 | 主色 | 辅色 | 注意 |
| --- | --- | --- | --- |
| 无声圣匣 | 冷灰、暗银、黑蓝 | 淡银白 | 安静、空、实验室感 |
| 低语田野 | 枯黄、暗褐、腐红 | 暗绿、铁锈红 | 不要过亮，不要田园 |
| 神血银 | 银白、半透明青灰 | 冷蓝高光 | 必须与血肉红形成对比 |
| 胃囊/饥饿 | 暗红、深肉色 | 黑紫、湿亮高光 | 少量高饱和用于弱点和反馈 |

## 角色资产

| 资产 | 首版需求 | 识别点 | 动画优先级 |
| --- | --- | --- | --- |
| 无韵回响 | 小体型俯视角角色 | 银白身体、胸口胃纹、无明确脸 | 移动、攻击、受击、死亡 |
| 收藏家 | 半身像或场景剪影 | 银灰长袍、记录器具、像医生不像恶魔 | 首版可静态 |
| 空腹者 | 基础敌人 | 干瘪腹部、大嘴、背肉粮篓 | 移动、扑咬、受击、死亡 |
| 饥民农夫 | 中距离敌人 | 农具、破布、银雾面罩 | 移动、投掷、受击、死亡 |
| 饥饿稻草人 | 精英敌人 | 稻草身、牙齿麦穗、腹部空洞 | 召唤、突刺、受击、死亡 |
| 谷仓王 | Boss | 谷仓门脸、巨大腹腔、胃囊弱点 | 三阶段、召唤、吞噬、死亡 |

## UI 与图标

| 资产 | 首版形式 | 说明 |
| --- | --- | --- |
| HP 条 | 暗红生命条 + 银边 | 不要科幻 HUD |
| 胃囊状态 | 开启/闭合两个小图标 | 对应击杀回血是否可用 |
| 溢血转刃 | 短时红银刀锋图标 | 满血溢出反馈 |
| 记忆晶片 | 半透明银白碎片 | 叙事收集物 |
| 房间标题牌 | 短句 + 地名 | 氛围强，但不要长诗 |

## 首版占位标准

在正式资产前，可以继续使用几何图形，但必须保留以下识别：

- 玩家是银白圆形，胸口有红色胃纹。
- 敌人按颜色和轮廓区分。
- Boss 胃囊暴露必须有明显红色弱点。
- 危险地面必须有红色半透明范围。
- 圣匣必须与田野在色彩上强烈区分。

## 验收标准

- 截图一眼能看出不是普通农场，而是“血肉麦田”。
- 玩家、敌人、危险地面、Boss 弱点在 2 秒内可分辨。
- 神血银和胃囊红色形成稳定视觉语言。
- UI 不遮挡战斗空间。
- 所有资产都服务低语田野，不提前画十城邦全量。
