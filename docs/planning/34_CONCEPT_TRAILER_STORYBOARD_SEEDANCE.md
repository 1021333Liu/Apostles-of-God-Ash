# 《神烬使徒 / Apostles of God-Ash》概念预告片分镜与 Seedance 提示词

版本：2026-06-30  
用途：课堂展示概念片，不以完整实机演示为目标。  
建议成片时长：2:20 - 2:50。  
画幅：16:9。  
整体气质：低饱和乡土暗黑、干燥旧纸、脏银、泥土、暗红血肉点缀、档案/圣匣/胃囊意象。  

## 00 制作原则

这支片子不是玩法长录屏，而是 vertical slice trailer / concept trailer：让观众相信“这个游戏将会是什么样”。

核心策略：
- 多用电影镜头、场景、角色、道具和 UI 符号。
- 少放长时间实机操作。
- Godot 画面只作为可信度插片：对话框、技能栏、骰子、日志归档、场景切换。
- 视频演出负责质感，Godot 截图负责证明这是游戏项目。
- 每段视频结尾尽量停在一张能接到当前 Godot 背景/角色站位的构图上。

负面方向：
- 不要赛博、二次元、高饱和、塑料骰子、手游按钮 UI。
- 不要做成纯 MV，不要过度炫技镜头。
- 不要把怪物做成普通丧尸/恶魔。
- 稻草人不要人腿，不要过分血腥恐怖。
- 农夫拿旧镰刀，但不是死神镰刀/魔法镰刀。

## 01 成片结构

总时长建议：约 160 秒。

1. 开场世界观：0:00 - 0:28
2. 主角与收藏家：0:28 - 0:55
3. 低语田野与四个样本：0:55 - 1:45
4. 骰子/技能/圣匣机制蒙太奇：1:45 - 2:18
5. 谷仓王与结尾标题：2:18 - 2:45

课堂展示如果时间紧，可以剪成 110 秒版本：保留镜头 1、2、3、5、7、10、12、14、16。

## 02 旁白稿

建议使用低声、冷静、像档案记录一样的男声或中性声线。不要热血，不要喊口号。

完整旁白：

> 这里不是田野。  
> 这里是神留下的胃囊。  
>
> 饥饿被写进名单，粮食被做成秩序。  
> 没有人真正死去，他们只是被登记、分配、再次吞咽。  
>
> 我在圣匣中醒来。  
> 收藏家说，我不是第一个样本，也不会是最后一个。  
>
> 空腹者还抱着碗，等一口永远不会来的救济。  
> 农夫守着田垄，把名字刻进脏银登记牌。  
> 稻草人站在血麦边界，没有腿，也从不离开。  
> 谷仓王坐在失败的救济里，像一座学会进食的仓库。  
>
> 在这里，攻击不是勇气。  
> 防御不是退缩。  
> 每一次掷骰，都是向圣匣交出一份证词。  
>
> 失败会被保存。  
> 记忆会被归档。  
> 下一次醒来时，收藏家会问：  
>
> 这一次，你带回了什么？

短版旁白：

> 这里不是田野。这里是神留下的胃囊。

> 每一次掷骰，都是向圣匣交出一份证词。


Seedance 建议参数：
- 单镜头时长：6-10 秒。
- 运动强度：低到中。
- 镜头语言：slow push in、slow lateral tracking、low angle、top-down archive table、close-up insert。
- 生成策略：每个镜头先出 2-3 个版本，优先选构图稳定、角色不变形、无乱码文字的版本。
- 字幕和 UI 不要让视频模型生成，后期剪辑里加。

## 07 BGM 与音效设计

BGM 方向：
- 低频 Drone。
- 稀疏弦乐。
- 旧风琴/低音提琴/擦弦质感。
- 中后段加入缓慢鼓点，但不要热血。

音效清单：
- 圣匣：金属锁扣、木盒开合、玻璃瓶碰撞。
- 档案：纸页、钢笔、印章、蜡封裂开。
- 田野：干草风声、远处木板、低频空气。
- 骰子：骨骰/石骰滚动，不要塑料骰子。
- 攻击：短促金属刮擦、布料撕裂。
- 重击：木梁断裂、钝重冲击。
- 防御：旧铁共振、厚书合上。
- 大招：蜡烛燃烧、低频胃鸣、纸张燃尽。
- 归档：纸片入匣、锁扣合上。

## 08 剪辑节奏

推荐节奏：
- 0:00 - 0:55：慢，建立世界观。
- 0:55 - 1:45：中速，逐个样本登场。
- 1:45 - 2:18：快，机制蒙太奇。
- 2:18 - 2:45：慢，谷仓王与标题收束。

转场：
- 档案纸擦除。
- 圣匣黑场。
- 蜡烛闪灭。
- 骰子落点切换。
- 印章落下切换。

字幕：
- 不要每句话都上字幕。
- 旁白关键句可以上屏：
  - `这里不是田野。`
  - `这里是神留下的胃囊。`
  - `每一次掷骰，都是一份证词。`
  - `这一次，你带回了什么？`
