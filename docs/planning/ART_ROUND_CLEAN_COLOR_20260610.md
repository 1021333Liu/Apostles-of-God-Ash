# ART_ROUND_CLEAN_COLOR_20260610

Branch: `feature/art-iteration`

## Direction

This round replaces the earlier heavy / ornate concept direction with a cleaner visual target:

- richer color blocks
- simple readable linework
- light oil-and-ink wash texture
- lower detail density
- more negative space
- no thick impasto rendering
- no Diablo-like gothic grime
- no over-decorated costume or UI framing

The map direction must still clearly read as Low Whispering Field, so the room concept keeps wheat rows, field boundaries, irrigation color, and a barn-gate shape.
