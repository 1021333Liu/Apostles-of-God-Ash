# 《神烬使徒》项目世界观与垂直切片规划

## 1. 项目定位

《神烬使徒》是一款黑暗奇幻动作肉鸽游戏。玩家扮演一个没有固定人格、不会被共鸣瘟疫同化的“无韵回响”，反复进入十座被神明残响污染的共鸣城邦，击败共鸣使徒，收集神之残骸，并用这些残骸构筑每一轮战斗风格。

游戏表层目标是“击败十位使徒，重谱终曲，拯救世界”。真正的核心问题是：

> 如果所有情感、能力和记忆都来自外部装填，你还能算是一个“人”吗？

本项目的核心体验不是单纯刷数值，而是让玩家在每一次死亡、每一次残骸选择、每一次击败 Boss 后，都感觉自己正在拼装一个越来越强、也越来越不稳定的“自我”。

## 2. 核心关键词

- 类型：动作肉鸽、俯视角战斗、Boss 驱动、Build 构筑。
- 美术基调：黑暗奇幻、神性残骸、认知瘟疫、华丽腐败、悲剧救赎。
- 叙事基调：前期杀死暴君，中期理解暴君，后期害怕自己成为暴君。
- 玩法核心：死亡循环、残骸装填、城邦探索、使徒 Boss、侵蚀倾向、结局分歧。
- 世界核心：旧神已死，世界被“万音”污染，人类靠主动感染较轻的疯病来抵抗更重的死病。

## 3. 世界观总览

### 3.1 万音

“万音”不是音乐，也不是神谕，而是一种宇宙层面的异常噪声。它穿透物质、意识、记忆与信仰，让所有存在都被迫寻找一个“频率”来维持自我。

旧神曾经是世界的调律者。祂们用自身神格过滤万音，让生命拥有稳定的感知、死亡、语言和历史。后来万音失控，旧神被撕裂，死前的意志、恐惧、欲望和权能化作无数“本源回响”，落入世界。

这场灾难被称为“万音崩毁”。

### 3.2 共鸣瘟疫

共鸣瘟疫是一种认知感染。它不是简单的病毒，而是某种强大回响捕获人的心智，使人逐渐变成该情感或执念的载体。

感染分三阶段：

1. 染响期：个体开始出现偏执行为，但仍保留自我。
2. 谐同期：自我意识与回响融合，外表开始异化。
3. 静默期：个体结晶化，意识被封在晶体中，永远演绎同一种回响。

静默者不是尸体。他们仍然“活着”，但已经不再是人。

### 3.3 神血银 / 锈银

神血银是旧神残骸在人间化作的半透明银白液态金属，又被称为“锈银”。它可以储存回响，是这个世界最珍贵、也最危险的资源。

锈银用途：

- 储存回响。
- 维持城邦领域。
- 强化武器、防具、义肢、面具、圣铃、乐器和刑具。
- 延缓共鸣感染。

副作用：

- 长期接触会产生幻听和“频感”。
- 依赖银雾会加速结晶化。
- 对普通人而言，它既是奢侈品、药物，也是毒品。

### 3.4 共鸣城邦

人类发现无法彻底抵抗万音后，选择主动拥抱一种较轻的回响，把它作为“主频”，以此抵抗其他回响。

于是十个以特定回响为核心的城邦建立起来。每座城邦都有一位与主频彻底同步的统治者，称为“共鸣使徒”。

使徒拥有近乎神的力量，但代价是失去完整人性。他们既是统治者，也是某道回响的人形牢笼。

## 4. 主角与主要角色

### 4.1 无韵回响

玩家角色。旧神碎裂时诞生的一块特殊碎片，没有固定频率，没有稳定人格，也不会被任何回响同化。

核心特质：

- 免疫共鸣：不会被共鸣瘟疫吞噬。
- 容器属性：可以装填多个神之残骸。
- 记忆读取：吸收残骸时读取旧神、使徒和静默者的记忆。
- 无韵：在使徒感知中像一块沉默的石头。

主角初始动机不是复仇或救世，而是“获得自我”。随着残骸装填，他逐渐意识到自己可能不是第一次被唤醒，也可能只是收藏家培养新神的容器。

### 4.2 收藏家

导师、复活机制维护者、残骸鉴定者。前期像可靠引导者，中期像医生，后期像实验记录员，终局揭示其真实目的：培养一个能承载万音的新神容器。

他的逻辑不是单纯邪恶。他相信只要新神诞生，世界就能从崩坏变成有序疯狂。

### 4.3 静默证人

未完全结晶化的静默者，记得多个版本的历史。每次说话都会掉落晶片，因此每条证词都有代价。用于提供与收藏家不同的真相。

### 4.4 断弦者领袖

反对一切调律的人。主张杀死使徒、摧毁神血银、让人类重新面对真实死亡。她可能过于激进，但最早指出收藏家的问题。

### 4.5 银誓民代表

普通幸存者立场。他不关心终极真相，只想让家人明天还能吸到银雾。用于提醒玩家：摧毁系统也会伤害被系统维持的人。

## 5. 十座共鸣城邦

每座城邦应包含：

- 一种核心回响。
- 一套社会规则。
- 一种神血银使用方式。
- 一类普通感染者。
- 一位共鸣使徒 Boss。
- 一块关键神之残骸。
- 一个关于旧神或万音的真相。

### 5.1 低语田野

- 核心回响：暴食。
- 视觉关键词：血肉麦穗、牙齿果实、肠道灌溉渠、饥饿稻草人、会咀嚼的谷仓。
- 社会规则：所有人都必须喂饱土地。
- 普通感染者：饥民农夫 / 空腹者。
- Boss：谷仓王。
- 残骸：神之胃囊。
- 玩法方向：击杀回血、过量治疗转化攻击力，但提高饥饿惩罚。
- 真相：旧神曾允许饥荒发生，以维持人类对神的依赖。

### 5.2 锈铁矿脉

- 核心回响：贪婪。
- 视觉关键词：无底矿井、血银矿车、肢体义械、硬币状眼球、会生长的矿脉。
- 社会规则：所有东西都可以被开采，包括骨头、名字、寿命和亲情。
- 普通感染者：抵押矿工。
- Boss：多肢者。
- 残骸：神之手骨。
- 玩法方向：抓取、牵引、处决、资源掠夺、复制精英能力。
- 真相：神血银不是神赐资源，而是旧神尸体仍在出血。

### 5.3 雾息牧场

- 核心回响：怠惰。
- 视觉关键词：孢子雪、软骨围栏、沉睡牲畜、人形茧、温柔腐烂的阳光。
- 社会规则：痛苦来自清醒，幸福来自放弃。
- 普通感染者：幸福活尸。
- Boss：梦蛹夫人。
- 残骸：神之肺叶。
- 玩法方向：毒雾、孢子、区域控制、减速、持续伤害。
- 真相：很多人并不想被拯救，他们知道现实更痛苦，因此主动选择幻梦。

### 5.4 狂欢集市

- 核心回响：色欲，指向对欲望本身的欲望。
- 视觉关键词：永夜灯会、面具、红丝帐、镜屋、畸形舞者、融化香水河。
- 社会规则：真实身体不重要，被渴望的形象才重要。
- 普通感染者：彩衣伶人。
- Boss：嬗变小丑。
- 残骸：神之面皮。
- 玩法方向：变形、诱饵、魅惑、闪避残影攻击。
- 真相：旧神也需要信徒的爱，神圣的一部分来自被凝视和被渴望。

### 5.5 焚书议院

- 核心回响：傲慢。
- 视觉关键词：圆形图书馆、尖叫书页、悬浮公式、黑焰灯塔、书匣头颅。
- 社会规则：无知是罪，错误是罪，怀疑也是罪。
- 普通感染者：书匣学者。
- Boss：全知者。
- 残骸：神之脑髓。
- 玩法方向：预判、反击、冷却缩减、技能连锁。
- 真相：万音可能曾被旧神主动研究甚至引来。

### 5.6 公正天秤 / 天平回廊

- 核心回响：嫉妒。
- 视觉关键词：巨型审判庭、镜面地砖、悬空刑架、称量心脏的天秤。
- 社会规则：每个人都可以审判别人，只要证明别人拥有自己没有的东西。
- 普通感染者：告密陪审员。
- Boss：镜像魔女。
- 残骸：神之眼。
- 玩法方向：复制敌人技能、标记弱点、偷取增益、反制精英能力。
- 真相：有些使徒最初也是受害者，获得力量后才成为新的审判者。

### 5.7 静默军械库

- 核心回响：暴怒。
- 视觉关键词：兵工炉、骨制炮弹、铁处女战车、无声军鼓、断肢旗帜。
- 社会规则：和平是怯懦，伤口是勋章，死亡是晋升。
- 普通感染者：燃血士兵。
- Boss：战争铁砧。
- 残骸：神之心炉。
- 玩法方向：怒气、燃烧、低血增伤、霸体、风险换爆发。
- 真相：旧神死前曾命令人类互相残杀，以筛选能承受万音的种群。

### 5.8 无面剧场

- 核心回响：背叛。
- 视觉关键词：无观众剧院、血幕布、提线木偶、可替换面孔、后台迷宫。
- 社会规则：忠诚是一种表演，表演越完美，越有资格背叛。
- 普通感染者：无面演员。
- Boss：千面人。
- 残骸：神之声带 / 神之喉舌。
- 玩法方向：欺骗、分身、技能伪装、敌人误伤、仇恨改写。
- 真相：收藏家曾多次培养“救世容器”，但每一个都被抛弃或背叛。

### 5.9 苍白王座

- 核心回响：绝望。
- 视觉关键词：灰烬雪、白骨宫殿、熄灭太阳、无泪哭脸、被封存的钟声。
- 社会规则：希望是危险幻想。
- 普通感染者：苍白臣民。
- Boss：共鸣使徒·王。
- 残骸：神之冠骨。
- 玩法方向：统御、全局强化、稳定随机池、牺牲随机奖励换确定性。
- 真相：王不是万音源头，只是一道闸门。杀死王会打开通往万音核心的路。

### 5.10 最初回响

- 核心回响：无。
- 视觉关键词：绝对空白、虚空裂缝、失败容器、旧神意识碎片、玩家残影。
- 社会规则：不存在社会，只有宇宙错误本身。
- 最终存在：万音核心。
- 玩法方向：规则层面的最终战，部分残骸短暂失效，场景回放前九城选择。
- 终局问题：玩家决定调律世界的方式。

## 6. 肉鸽叙事系统

### 6.1 核心循环

玩法循环：

1. 玩家从无声圣匣选择残骸槽位和初始构筑。
2. 进入一座共鸣城邦。
3. 城邦根据核心回响生成房间、敌人、事件和临时祝福。
4. 玩家获得临时回响、神血银、记忆碎片和残骸强化。
5. 玩家击败 Boss、撤离、死亡或被迫回收。
6. 回到无声圣匣，保留部分资源、记忆和永久解锁。

叙事循环：

1. 收藏家称进入城邦为“采样”。
2. 城邦用自身频率试图给无韵者赋形。
3. 玩家吸收临时回响，借用某种执念的力量。
4. Boss 通过战斗证明该城邦的生存逻辑。
5. 死亡把战斗数据、痛苦和残响带回圣匣。
6. 圣匣更新残骸、NPC台词和世界真相。

### 6.2 死亡反馈

死亡不只是结算，而是叙事推进器。每次死亡至少触发一种反馈：

- 收藏家记录死因。
- 圣匣出现与死因相关的物理变化。
- Boss 下次提到玩家上次的死法。
- 解锁残骸记忆或静默者低语。
- NPC 对玩家的非人性产生恐惧或同情。
- 常用残骸增加侵蚀痕迹。

### 6.3 残骸系统

残骸不是装备，而是人格碎片。每块残骸应包含三层反馈：

- 机制层：攻击、防御、位移、召唤、控制或资源循环。
- 记忆层：使用次数、击杀条件或 Boss 组合触发旧神 / 使徒记忆。
- 侵蚀层：改变主角外观、圣匣细节、NPC反应和结局倾向。

槽位建议：

- 神之心：决定战斗节奏。
- 神之脑：决定规则理解。
- 神之手：决定主动交互。
- 神之眼：决定信息优势。
- 神之肺：决定空间控制。
- 神之声：决定敌我关系。
- 神之冠：决定终局规则。

## 7. 结局方向

### 7.1 新神

条件倾向：

- 高频使用高侵蚀残骸。
- 接受收藏家解释。
- 保留较多城邦规则。

结果：主角成为新神，十城稳定，共鸣瘟疫不再恶化，但所有生命都被固定在可管理的频率内。

情感定位：稳定但压抑。玩家赢了世界，却失去无韵的自由。

### 7.2 终曲寂灭

条件倾向：

- 摧毁多数城邦核心。
- 拒绝收藏家。
- 倾向断弦者。

结果：主角以自身为代价奏响绝对沉默，回响、神血银、静默者和旧神残念消散。文明归零。

情感定位：彻底但残酷。世界摆脱万音，也失去所有由万音维持的奇迹和记忆。

### 7.3 封印

条件倾向：

- 完成使徒完整记忆。
- 保留幸存者。
- 发现旧容器档案。
- 控制残骸侵蚀。

结果：主角把世界回响压入自己体内，封入神血银晶体并沉入地底。世界恢复平凡，主角永恒承受万音。

情感定位：最接近慈悲，但不是无代价的好结局。

## 8. 第一版垂直切片建议

第一版不要做十座城邦。建议只做一座：低语田野。

原因：

- 主题最直观：饥饿、吞噬、土地吃人。
- 美术识别度高：血肉麦田、牙齿果实、饥饿稻草人。
- 机制好落地：击杀回血、地面消化液、作物/尸体被 Boss 吞噬。
- 叙事好开场：谷仓王最初是为了拯救饥荒，容易建立“Boss不是纯坏”的调性。

### 8.1 Demo 目标

目标不是做完整游戏，而是验证这几个问题：

1. 黑暗奇幻动作肉鸽基调是否成立。
2. 近战战斗是否有手感。
3. 残骸是否能同时提供玩法和叙事。
4. 死亡回圣匣是否能变成叙事循环。
5. 谷仓王 Boss 是否能体现“曾经合理，如今怪物”的悲剧感。

### 8.2 Demo 场景结构

1. 无声圣匣：起点、收藏家、残骸装填、死亡回归。
2. 低语田野入口：教学房，普通空腹者。
3. 血肉麦田：基础战斗房，带咬人麦穗陷阱。
4. 肠道灌溉渠：移动受限房，地形和伤害区。
5. 饥饿谷仓：精英房，饥饿稻草人。
6. 谷仓王胃室：Boss 房。
7. 回到圣匣：展示残骸、记忆碎片、收藏家台词变化。

### 8.3 Demo 敌人

空腹者：

- 基础近战敌人。
- 行为：缓慢追踪，低血时加速。
- 视觉：干瘪腹部、异化嘴部、背着肉粮篓。

饥民农夫：

- 中距离敌人。
- 行为：投掷血肉种子，种子落地后生成短暂牙齿作物。
- 视觉：农具、破布、银雾面罩。

饥饿稻草人：

- 精英敌人。
- 行为：站桩召唤麦浪，短距离突刺。
- 视觉：稻草身体、牙齿麦穗、腹部空洞。

谷仓王：

- 第一版 Boss。
- 阶段一：召唤空腹者，吞噬场景作物回血。
- 阶段二：地面出现消化液，限制玩家走位。
- 阶段三：Boss 体型膨胀，攻击频率提高，但暴露胃囊弱点。

### 8.4 Demo 残骸与道具

神之胃囊：

- 机制：击杀回血；满血时治疗溢出转化为短时攻击力。
- 代价：受击后进入饥饿惩罚，短时间内无法再通过击杀回血。
- 记忆：谷仓王曾把自己的胃献给土地，希望“吃掉饥荒”。
- 侵蚀：主角胸口出现暗红胃纹，收藏家评价“容器开始理解饥饿”。

锈银短刃：

- 初始武器。
- 机制：近战三连，第三击击退。
- 叙事：由神血银锻成，能切割回响，但每次挥动都在采样敌人残留频率。

静默者晶片：

- 叙事道具。
- 机制：解锁一段文本或收藏家记录。
- 用途：让玩家理解普通居民也参与维持城邦制度。

## 9. 组员分工建议

### 9.1 程序

优先任务：

- Godot 项目骨架。
- 俯视角移动与近战攻击。
- 房间切换。
- 敌人状态机。
- Boss 阶段切换。
- 残骸道具系统。
- 死亡回圣匣流程。

第一阶段不要急着做复杂随机地图。先把固定房间打磨到可玩。

### 9.2 美术

优先资产：

- 主角“无韵回响”基础形态。
- 收藏家立绘 / 半身像。
- 无声圣匣场景概念。
- 低语田野地块。
- 血肉麦穗。
- 牙齿果实。
- 空腹者。
- 饥民农夫。
- 饥饿稻草人。
- 谷仓王。
- 神之胃囊图标。
- 锈银短刃图标。

美术关键词：银白液态金属、暗红血肉、腐败麦田、神性器官、华丽但不干净。

### 9.3 叙事

优先文本：

- 开场收藏家台词。
- 死亡后收藏家台词 5-8 条。
- 低语田野场景短句。
- 空腹者 / 饥民农夫 / 饥饿稻草人图鉴。
- 谷仓王 Boss 开场、半血、死亡台词。
- 神之胃囊记忆碎片 3 条。
- 静默者晶片文本 5 条。

写作原则：

- Boss 曾经有合理起点。
- 居民不是单纯受害者，也可能是制度维护者。
- 收藏家早期不能明显邪恶。
- 每段文本都要服务世界观或玩法，不写空泛诗句。

### 9.4 策划

优先表格：

- 房间清单。
- 敌人参数。
- Boss 阶段设计。
- 残骸机制表。
- 掉落与奖励节奏。
- 死亡反馈触发条件。
- Demo 验收标准。

## 10. 第一阶段验收标准

Demo 至少应满足：

- 玩家能从无声圣匣进入低语田野。
- 至少 4 个战斗房 + 1 个精英房 + 1 个 Boss 房。
- 至少 3 种敌人。
- 至少 1 个完整 Boss。
- 至少 1 个残骸能改变战斗方式。
- 死亡后回到圣匣，并触发不同台词或记录。
- 击败 Boss 后获得神之胃囊与一段记忆。
- 玩家能看懂：这个世界不是单纯打怪，而是在收集别人的执念来拼装自己。

## 11. 当前项目决策

- 《神烬使徒》应作为主项目世界观。
- 旧西游 Godot 原型可以保留为技术练习和手感测试场。
- 后续真正的项目内容应逐步迁移到《神烬使徒》的黑暗奇幻动作肉鸽方向。
- 第一版不要追求 10 城邦全量实现，只做低语田野垂直切片。
- 所有机制都必须能回答：“它为什么存在于这个世界？”
