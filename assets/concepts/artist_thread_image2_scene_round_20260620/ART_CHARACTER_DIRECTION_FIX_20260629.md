# 2026-06-29 角色方向修正：农夫、稻草人、谷仓王

给金荣俊 / 美术处理线程：

这是最新角色方向修正，优先级高于前面长文档里的旧描述。请后续角色图、动作帧、i2i prompt 都按这里执行。

## 1. 谷仓王继续深化

谷仓王仍然要继续做，不要停在当前版本。

核心方向：

- 不是纯怪物。
- 不是普通巨大肉团。
- 要保留“曾经开仓救人的王”的人类 / 制度残影。
- 身体可以融合谷仓木梁、粮仓门、钥匙、胃囊、救济失败的痕迹。
- 要能看出他曾经是人，或者曾经被人当成王 / 救济者。

建议元素：

```text
barn king, failed granary savior, old wooden barn beams, rusty granary key, famine relief memory, dark red stomach sac, tarnished crown-like barn hardware, human remnant, heavy tragic silhouette
```

负面：

```text
no pure monster, no generic demon, no full meat cave creature, no cyberpunk, no anime boss, no glossy horror game monster, no oversized side-view background face
```

## 2. 农夫必须拿镰刀

农夫方向修正：希望他拿着镰刀。

核心方向：

- 农夫是制度执行者和受害者混合。
- 必须带旧镰刀，作为攻击动作和掉落物识别点。
- 镰刀不是华丽武器，应该是旧农具、锈迹、泥土、长期劳作痕迹。
- 农夫仍然干瘦、疲惫、低饱和、乡土，不要变战士。

建议元素：

```text
famine farmer holding an old sickle, rusted farm tool, rough cloth, mud stains, exhausted rural worker, famine registry enforcer, low saturation, dry old-paper texture
```

动作要求：

- `idle`：一手垂着或扶着镰刀，身体疲惫。
- `attack`：镰刀前挥 / 低位横扫，农具感明显。
- `hurt`：镰刀仍在手里或差点脱手。
- `talk`：镰刀下垂，像一边登记一边喃喃。

负面：

```text
no sword, no spear, no fantasy warrior weapon, no clean polished scythe, no anime farmer, no zombie soldier
```

## 3. 稻草人不要太恐怖，也不要加腿

饥饿稻草人方向修正：不要太恐怖，不要加腿。

核心方向：

- 必须是一眼读成“稻草人”的结构。
- 不要给稻草人加腿。
- 下半身应是木杆、插地木桩、绑扎稻草、布条垂落。
- 不要画成穿破衣的人。
- 不要画成会跑的人形怪。
- 恐怖程度要克制，偏荒凉、守卫、被遗忘，而不是跳脸怪。

建议结构：

```text
hungry scarecrow, no legs, wooden stake lower body, crossbar arms, straw bundle torso, torn old cloth, tied fabric strips, quiet field guardian, dry wheat, low saturation rural horror, not too scary
```

动作要求：

- `idle`：固定在木杆上轻微晃动，布条和稻草动。
- `attack`：麦秆 / 木杆 / 血麦刺出，不是迈腿冲刺。
- `defend`：稻草和破布收拢。
- `hurt`：木杆震动、稻草散落。
- `talk`：头部或布条轻动，像风里低语。

负面：

```text
no legs, no walking humanoid, no human pants, no running pose, no extreme gore, no jump-scare face, no Halloween cartoon scarecrow, no anime
```
